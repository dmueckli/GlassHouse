# GlassHouse

Project for monitoring temperature and humidity for plants and sending the data to a webserver. 

## Partlist:
1. ESP32 
2. DHT22 Temperatur & Humidity Sensor
3. SoilMoisture Sensor (any 3.3V or 5V will do)
4. 128\*64 TFT Display
